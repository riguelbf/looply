# reinstall

Reinstall looply for selected hosts using the current pack and locale

## Arquivo de origem

- `src/commands/reinstall.ts`

## Options

- `--host <host>`: Target host list such as codex,claude
- `--scope <scope>`: Installation scope such as project or global
- `--dir <dir>`: Target directory for project scope reinstall (defaults to current directory)
- `--source-root <dir>`: looply source directory that contains packs/
- `--locale <locale>`: Override output locale such as pt-BR or en
- `--yes`: Skip confirmation
