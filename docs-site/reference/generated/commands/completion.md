# completion

Generate or inspect shell completion for the looply CLI

## Arquivo de origem

- `src/commands/completion.ts`

## Options

- `--shell <shell>`: Shell name such as bash or zsh
- `--install`: Print install instructions together with the generated completion
- `--dir <dir>`: Target directory for repository-aware completion values (defaults to current directory)
- `--index <index>`: Current token index
