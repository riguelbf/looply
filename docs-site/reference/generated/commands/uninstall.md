# uninstall

Remove looply files for one or more installed hosts

## Arquivo de origem

- `src/commands/uninstall.ts`

## Options

- `--host <host>`: Target host list such as codex,claude
- `--scope <scope>`: Installation scope such as project or global
- `--dir <dir>`: Target directory for project scope uninstall (defaults to current directory)
- `--yes`: Skip confirmation
