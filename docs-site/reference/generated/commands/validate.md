# validate

Validate packs, references and contracts

## Arquivo de origem

- `src/commands/validate.ts`

## Options

- `--source-root <dir>`: looply source directory that contains packs/
- `--harness`: Also validate the installed harness in the target project
- `--dir <dir>`: Target project directory for --harness (defaults to current directory)
- `--host <host>`: Host to validate the harness against (codex or claude)
