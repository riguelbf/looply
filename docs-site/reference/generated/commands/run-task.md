# run-task

Register a manual task execution inside a feature workflow

## Arquivo de origem

- `src/commands/run-task.ts`

## Options

- `--dir <dir>`: Target directory for the feature state (defaults to current directory)
- `--reason <reason>`: Reason for the manual task execution
- `--source-root <dir>`: looply source directory that contains packs/
