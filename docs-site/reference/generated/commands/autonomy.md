# autonomy

Plan a host-driven autonomous cycle for a feature workflow

## Arquivo de origem

- `src/commands/autonomy.ts`

## Options

- `--dir <dir>`: Target directory for the project state (defaults to current directory)
- `--host <host>`: Host to target when deriving the cycle
- `--json`: Print the autonomy cycle as JSON
- `--source-root <dir>`: looply source directory that contains packs/
