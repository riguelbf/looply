# check-updates

Check whether installed hosts have updates available

## Arquivo de origem

- `src/commands/check-updates.ts`

## Options

- `--host <host>`: Target host list such as codex,claude
- `--scope <scope>`: Installation scope such as project or global
- `--dir <dir>`: Target directory for project scope update check (defaults to current directory)
- `--source-root <dir>`: looply source directory that contains packs/
