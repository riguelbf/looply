# replay

Replay a feature workflow from a stage, agent, task or artifact checkpoint

## Arquivo de origem

- `src/commands/replay.ts`

## Options

- `--dir <dir>`: Target directory for the feature state (defaults to current directory)
- `--reason <reason>`: Reason for replay
- `--source-root <dir>`: looply source directory that contains packs/
