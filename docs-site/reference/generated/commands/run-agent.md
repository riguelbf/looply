# run-agent

Register a manual agent intervention inside a feature workflow

## Arquivo de origem

- `src/commands/run-agent.ts`

## Options

- `--dir <dir>`: Target directory for the feature state (defaults to current directory)
- `--reason <reason>`: Reason for the manual intervention
- `--source-root <dir>`: looply source directory that contains packs/
