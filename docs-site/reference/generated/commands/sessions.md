# sessions

Inspect and manage looply session links for multi-session workflows

## Arquivo de origem

- `src/commands/sessions.ts`

## Options

- `--dir <dir>`: Target directory for session links (defaults to current directory)
- `--workflow <workflow>`: Workflow name, such as story-to-production
- `--last-command <command>`: Last command used for this session
