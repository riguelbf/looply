# ledger

Manage the context ledger for feature workflow decision memory

## Arquivo de origem

- `src/commands/ledger.ts`

## Options

- `--dir <dir>`: Target directory (defaults to current directory)
- `--summary-only`: Read only the context summary
- `--rationale <text>`: Rationale behind the decision
- `--constraints <text>`: Constraints discovered
- `--risks <text>`: Risks identified
