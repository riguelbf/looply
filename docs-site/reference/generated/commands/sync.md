# sync

Synchronize managed artifacts to an installed host

## Arquivo de origem

- `src/commands/sync.ts`

## Options

- `--host <host>`: Target host list such as codex,claude
- `--scope <scope>`: Installation scope such as project or global
- `--dir <dir>`: Target directory for project scope sync (defaults to current directory)
- `--source-root <dir>`: looply source directory that contains packs/
- `--yes`: Skip confirmation and apply available updates
