# Desktop App Tech Spec

## Purpose

Definir uma futura camada desktop para o LOOPLY que amplie a acessibilidade do produto sem substituir a operacao `CLI-first`.

## Positioning

- o desktop app e um companion local
- a CLI continua como superficie principal
- o app prioriza descoberta, visualizacao de workflow, docs e contexto

## Recommended Stack

- Electron
- TypeScript
- React
- Tailwind

## MVP Direction

- overview do projeto
- visualizacao de workflows e features
- leitura de docs e catalogo
- contexto do projeto
- proximos comandos orientados por host

## Prerequisites

- `status` mais rico
- snapshots consumiveis pela UI
- workflows novos completos
- documentacao oficial alinhada
