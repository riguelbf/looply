# ADR 0002 - Workflow Handoff

## Status

Accepted

## Context

Workflows precisam ser legiveis por humanos e trataveis por ferramenta.

## Decision

Workflow tera handoff explicito no frontmatter e no corpo do Markdown.

## Consequences

- validacao pode checar consistencia de transicoes
- o host recebe contexto de passagem de forma mais clara
- o workflow fica menos ambiguo para agentes e pessoas
