# Release 1

## Theme

Artifact platform foundation

## Scope

- contratos canonicos
- validacao
- pack `engineering-base`
- publicacao inicial para hosts
- CLI de bootstrap e manutencao

## Out of Scope

- runtime proprio de tasks
- payments squad
- reverse engineering
- upgrade semantico entre versoes

## Exit Criteria

- `engineering-base` valido
- manifesto de publicacao definido
- CLI com `validate`, `install`, `sync`, `doctor`, `list`, `inspect`
- estrutura de customizacao local definida
