# Definition Of Done

- [ ] objetivo claro
- [ ] escopo delimitado
- [ ] dependencias registradas
- [ ] contrato de entrada e saida revisado
- [ ] impacto em persistencia e integracoes avaliado
- [ ] testes relevantes atualizados
- [ ] observabilidade e operacao consideradas quando necessario
- [ ] riscos destacados
- [ ] proximo handoff definido
